# v2.0.0 — 标签系统 + Token 用量 + 多语言

在 v1.0.0 基础上新增标签管理、Token 用量统计、多语言 UI 切换三大功能。

---

## 🆕 v2 新增功能

### 🔖 标签系统

- **文章打标签**：阅读页工具栏 `#` 按钮，弹出标签面板
- **手动输入**：自由创建标签，支持大小写不敏感匹配已有标签
- **AI 推荐**：面板打开自动调用 LLM 分析文章标题+摘要，生成 3-5 个标签建议
- **NLP 兜底**：从标题提取大写实体词作为本地建议，AI 失败时静默降级
- **侧边栏标签筛选**：侧边栏分段切换（订阅源/标签），点击标签筛选文章
- **多选 + OR/AND 模式**：可同时选多个标签，切换匹配任一/匹配所有
- **标签搜索**：实时过滤标签列表
- **右键菜单**：重命名、删除标签
- **标签库管理**：设置 > 标签 → 搜索/过滤/别名/合并/批量删除
- **批量打标签**：设置 > 批量标签 → 选择时间范围 → AI 批量分析
- **标签颜色**：每个标签自动分配固定颜色圆点

### 📊 Token 用量统计

- **自动记录**：每次摘要/翻译/标签推荐完成后自动写入数据库
- **用量面板**：设置 > 用量 → 折线图 + 统计卡片
- **多维对比**：按 Provider、Model、Agent 类型分别统计
- **数据管理**：可配置保留天数（30/90/180/365/永久），支持清除过期/全部

### 🌐 多语言 UI

- **中英切换**：设置 > 外观 > 语言下拉，即时生效无需重启
- **跟随系统**：默认跟随操作系统语言
- **320 条翻译**：覆盖全部界面文字、提示词、按钮
